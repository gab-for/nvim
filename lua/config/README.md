# Configuration files

## Mapping
- `<Space>c` : clear selected
- `<Space>t` : open tree file explorer

## LSP : Required installation
### pyright (Python)
```
# installation
npm i -g pyright
# check install
pyright --version
```

### bashls (Bash)
```
# installation
npm i -g bash-language-server
# check install
bash-language-server --version
```

### gopls (Go)
```
# installation
go install golang.org/x/tools/gopls@latest
# check install
bash-language-server --version
```
