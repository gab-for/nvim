# Neovim Configuration
(Relatively) Light weight neovim configuration using the lazy package manager, for 

- `config/` : configuration files directory
    - `lazy.lua` : lazy package manager settings
    - `lsp.lua` : LSP server settings
    - `mappings.lua` : keybinding settings
    - `settings.lua` : general neovim settings set from [kickstart.nvim](https://github.com/nvim-lua/kickstart.nvim)
- `plugins` : directory with every plugins configuration
    - `alpha.lua`  : starting menu ([github](https://github.com/goolord/alpha-nvim))
    - `autoclose.lua` : auto pairs and close brackets ([github](https://github.com/m4xshen/autoclose.nvim))
    - `blink.cmp` : completion plugins ([github](https://github.com/saghen/blink.cmp))
    - `catppuccin` : pastel color theme ([github](https://github.com/catppuccin/nvim))
    - `nvim-colorizer` :  highlighter for written color ([github](https://github.com/catgoose/nvim-colorizer.lua))
    - `nvim-tree` : file explorer tree ([github](https://github/nvim-tree/nvim-tree.lua))
